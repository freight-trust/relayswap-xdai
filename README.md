# RelaySwap xDai

> Enterprise Finance

## Usage 
* Injected web3 Provider (e.g. nifty, metamask) configured for xDai network [chainid: 100] with at least `0.01` xDAI && at least `1 $EDI`

### Demo

**Java 8+ Required**

> PKI Fingerprint

* Deployment Script

```bash
. ./init-api.sh
```

* Start demo.

`java -jar relayswap-0.1.0.jar`

* Note down the address and transaction identifier.
